# Solar Garlic

Website for Solar Garlic, a Denver jam band.

# Local Development

## With vscode

To get started developing the site with vscode, you will need:

1. To install [vscode](https://code.visualstudio.com/download)
2. To install the [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) and [LiveServer](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
3. Git installed and an optional git extension for vscode, such as [Gitlens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens).

You can run the project with LiveServer by right clicking `index.html` and selecting the `Open With Live Server` option. Changes to the project should now be viewable in a browser in real-time.
